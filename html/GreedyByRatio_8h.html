Algoritmo guloso baseado na razão lucro/peso para o problema da mochila.

Ordena os paletes pela razão lucro/peso e seleciona-os enquanto a capacidade permitir. É um algoritmo aproximado, não garante a solução ótima.

Parameters
palletsVetor com todos os paletes disponíveis.
truckCapacityCapacidade máxima do camião.
usedItems2Array booleano que indicará quais paletes foram selecionados.
Returns
unsigned int Lucro total obtido pela seleção gulosa.

@complexity Tempo: O(n log n) devido à ordenação dos paletes. Espaço: O(n) para o vetor auxiliar.
